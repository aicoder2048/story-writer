# 人物小传

## 主要角色

**小星（女主）**

一颗住在繁星园角落的小星星，有着淡银色的身体和五个圆润的角。与其他星星不同，她的身体不会发出耀眼的光芒，只有一种若隐若现的柔和微光。她的眼睛大大的，像是装着两个小月亮，却总是带着怯怯的神色。说话声音很轻，像风吹过云朵的声音。她胆小、自卑、总是躲在别人看不到的地方。因为不会发光，她从小就被同伴们嘲笑，渐渐开始怀疑自己是不是真正的星星。但在她胆小的外表下，藏着一颗温柔善良的心——她会偷偷帮迷路的流星找到方向，会把自己的云朵床让给累了的萤火虫休息。当朋友遇到危险时，她愿意不顾一切地去保护。这趟冒险让她发现，真正的光芒不在于有多耀眼，而在于能够温暖多少心灵。她学会了接纳自己，不再为"不同"而自卑，因为她的"不同"，正是她最珍贵的地方。

**萤火虫阿亮（伙伴）**

一只住在光明森林边缘的小萤火虫，圆圆的身体，大大的眼睛，尾巴上挂着一盏小小的灯。他的光芒在萤火虫中算是很微弱的，但他从来不觉得这是问题。"我的光虽然小，但每天晚上都会亮！"是他最常说的话。他热情、乐观、话特别多，像一个移动的小太阳。他第一次见到小星时，并没有因为她不发光而惊讶或嘲笑，而是真心觉得她"很特别"。在冒险中，他总是鼓励小星，用自己微弱却坚定的光芒为她照亮前路。他教会小星一个道理：光芒的大小不重要，重要的是永远不熄灭。当他为保护小星而耗尽光芒时，正是他的牺牲精神唤醒了小星内心的力量。最后，被小星的温柔之光照亮后，他的光芒反而比从前更亮了——因为他的心里，装进了更多的爱。

**月亮婆婆（导师）**

银河中最慈祥的存在，每天晚上挂在天空最中央，守护着所有星星。她的身体是温润的乳白色，脸上永远带着温和的微笑，像奶奶看着孙子的表情。她知晓很多古老的秘密，却从来不直接告诉答案，而是用充满诗意的话语给予启发。她一直关注着小星，知道这个不会发光的孩子有着最特别的潜力。"真正的光芒，不在外表，而在心里。"是她对小星说的最重要的话。当黑夜先生来袭时，她选择相信小星，把拯救大家的任务交给这个"最不可能"的人选。她相信，正是因为小星的"不同"，她才能成为唯一不受黑暗影响的存在。她是智慧的象征，代表着爱与信任的力量。

## 反派角色

**黑夜先生（主要反派）**

一团浓重的黑色影子，没有固定的形状，有时像一只巨大的手，有时像一条无边的河流。他的声音冷冷的，像是从深井底传上来的回音。他的眼睛是两个小小的暗红色光点，看人的时候让人从心底发寒。他曾经也是一颗星星，叫做"暗星"，因为光芒不够亮而被其他星星嘲笑。与小星不同，他没有选择接纳自己，而是充满了嫉妒和怨恨。他开始讨厌光明，讨厌所有发光的存在，渐渐被内心的黑暗吞噬，变成了现在的样子。他想要熄灭所有的光芒，让整个世界陷入永恒的黑暗。但他内心深处，还残存着一点点当初作为星星的善意。当小星的温柔之光照到他时，那点善意被唤醒，他终于得到了解脱。他的结局不是被消灭，而是被救赎——这是童话式的正义。

## 重要配角

**老萤火虫**

一只年老的萤火虫，翅膀已经有些破旧，飞得很慢很慢。他住在黑暗河边的一棵老树上，曾经是光明森林的守护者，现在已经退休了。他的光芒虽然老旧，却从未熄灭过。他送给小星光芒种子时，眼中带着期许的光——他看出了小星身上的潜力。"只有拥有真正光芒的人，才能让种子开花。"他是第一个相信小星能够找到光芒的陌生人。

**暗影鱼**

黑夜先生的手下，一群由黑暗凝聚成的鱼形生物。它们藏在黑暗河里，会吞噬一切光芒。虽然是反派的手下，但它们并不可怕，更像是调皮的小怪兽。它们害怕光芒，即使是阿亮那么微弱的光也能吓退它们。在黑夜先生被救赎后，暗影鱼也变回了普通的小鱼，回到了河里快乐地游泳。

**光明之源**

一朵生长在光明森林深处的巨大花朵，是所有光芒的源头。她有着彩虹色的花瓣和金色的花蕊，会说话，声音像是叮咚的泉水。她见证了小星的成长，最后送给小星一片花瓣作为祝福。这片花瓣成为小星永远闪烁的标志，证明她找到了属于自己的光。

## 其他角色

**星星同学们**：住在繁星园的其他星星们，有的发金光，有的发银光，各有各的亮点。它们曾经嘲笑小星，但当小星拯救了大家后，它们真心地敬佩她，学会了尊重不同。

**云朵小姐**：住在小星家旁边的一朵白云，胖胖软软的，总是让小星躲在她身后。她是小星为数不多的朋友，温柔包容，会在小星难过时轻轻抱住她。
