# 人物小传

## 主要角色

**陈默(男主)**

27岁,斯坦福CS博士辍学创业者,身高178cm,戴黑框眼镜,常年T恤牛仔裤。他的电脑桌面永远是Terminal黑底白字,机械键盘的敲击声在深夜格外清脆。作为技术天才,他可以熬夜72小时优化一个算法,却无法在投资人面前讲出一个完整的商业故事。他的GitHub上有37个开源项目,Stack Overflow声望值排名前0.1%。

性格上是典型的理想主义者和完美主义者。"代码不会撒谎,但人会"是他的口头禅。他相信技术能改变世界,相信做正确的事最终会获得回报,哪怕这意味着要付出巨大代价。他有轻微的社交障碍,在人多的场合会不自觉地揉太阳穴,但和代码相处时却异常放松。

他的成长弧光是从技术至上的单纯,到理解商业妥协的必要,再到在理想与现实间找到自己的平衡点。最终他选择离开公司回到学术界,不是因为失败,而是因为认清了自己真正想要的是什么——不是成功,而是做正确的事。

**李明(联合创始人)**

28岁,陈默的大学室友,斯坦福MBA。他和陈默是互补型搭档:陈默懂技术不懂商业,他懂商业不懂技术。他擅长谈判、融资、讲故事,能在30秒内抓住投资人的兴趣点,能在董事会上据理力争。他总是穿着得体的商务休闲装,MacBook上贴满了各种创业公司的logo贴纸。

他是现实主义者。他同样有改变世界的梦想,但他更明白:只有活下来,才能改变世界。他可以为了公司生存做出妥协,可以在关键时刻牺牲完美换取速度,可以在友谊和公司利益间做出艰难选择。投票罢免陈默那一票,是他最痛苦的决定,但他认为这是正确的——至少对公司来说。

他的内心冲突在于:他知道陈默是对的,但他选择了另一条路。当他站在纳斯达克敲钟时,他会想起那个在车库里立下的誓言,会想起那个深夜和陈默的争吵。成功了,但失去了最好的朋友,这是他的代价。

**林悦(投资人)**

42岁,知名VC合伙人,曾投资过三家独角兽。她总是穿着剪裁得体的套装,手腕上是Apple Watch,谈话时会习惯性地看数据面板。她有敏锐的商业嗅觉,能在五分钟的demo里判断一个项目的价值,也能在term sheet里设置各种保护条款。

她不是反派,她只是一个专业的投资人。她的职责是让基金获得回报,她的判断标准是数据和市场,不是情怀和理想。她欣赏陈默的技术能力,但不认同他的理想主义:"创业不是做学术研究,是做生意。"她推动罢免陈默,不是因为私人恩怨,而是基于商业判断:陈默的决策会让公司死掉。

她代表了资本的逻辑和商业的规则。在她的世界里,没有对错,只有成功和失败。但在深夜独处时,她也会想:如果每个创业者都像陈默那样妥协,这个世界还会有创新吗?

**张云(竞争对手)**

35岁,BAT大厂出身,连续创业者。他擅长快速复制成功模式,用资源和速度碾压对手。他不是窃取代码的主谋——是陈默公司的离职员工主动投靠,他只是接受了这个"礼物"。在他看来,这是正常的商业竞争:"硅谷哪家公司不是相互挖人、借鉴创意?这就是市场规则。"

他和陈默是两个极端:陈默追求技术完美,他追求市场速度;陈默理想主义,他现实主义。他不是坏人,只是价值观不同。当他的产品出现医疗事故时,他也会反思:速度优先真的对吗?但商业的惯性已经无法停下。

**王教授(导师)**

65岁,斯坦福AI实验室教授,陈默的博士导师。白发苍苍,温和儒雅,办公室里堆满了论文和书籍。他是陈默学术上的启蒙者,也是精神上的导师。当陈默决定辍学创业时,他说:"去吧,学术界会一直在这里等你。但改变世界的机会,可能只有一次。"

当陈默失意回到斯坦福时,他没有说"我早就告诉过你",而是说:"你做了正确的事。有些东西比成功更重要。"他代表了学术界的价值观:追求真理,而非利益;追求完美,而非速度。他是陈默的精神支柱,也是故事中"另一种成功"的象征。

## 重要配角

**周萌(早期员工,离职投敌)**

26岁,公司第5号员工,后端工程师。她加入时相信公司的理想,但半年后发现:理想不能当饭吃。她有助学贷款要还,有父母要养,而公司的股票还不知道能不能兑现。当张云开出双倍薪资挖她时,她动摇了。她带走了部分代码,成为了"叛徒"。

她代表了普通员工的现实困境:不是每个人都有资本去坚持理想。她的离职,让陈默第一次意识到:创业不只是他和李明的梦想,还关系到几十个员工的生计。

**David(硅谷老炮,天使投资人)**

55岁,投资过Apple早期,经历过互联网泡沫。他给了陈默最早的10万美元天使投资,不是因为看好商业前景,而是因为"我当年也是这么过来的"。他见过太多理想主义者被现实打败,但依然相信:这个世界需要这样的傻瓜。

他在关键时刻给陈默建议:"保持初心,但也要学会生存。不是所有的妥协都是背叛。"他是陈默和商业世界之间的桥梁。

**小雨(产品经理,团队稳定剂)**

29岁,前微软产品经理,加入创业公司寻求改变。她是团队里少数几个既理解技术又理解用户的人。当陈默和李明争吵时,她总是试图调和:"我们的目标是一致的,只是路径不同。"

她代表了团队里的中间派:既认同陈默的理想,也理解李明的现实。她最终选择留下,因为她相信:这个团队,即使分裂,依然在做有意义的事。

**赵律师(公司法律顾问)**

45岁,专业的创业公司律师。他见过太多合伙人因为股权撕破脸,他的工作就是把这些潜在冲突用法律条文固定下来。当陈默被罢免时,是他解释了term sheet里的条款:"你签字的时候,就同意了这个可能性。"

他代表了冰冷的契约精神:商业世界里,情怀和友谊最终都要落到纸面上的条款。

## 人物关系网

**核心三角关系**:
- 陈默 ←→ 李明:从默契搭档到理念分裂,从亲密战友到对立董事
- 陈默 ←→ 林悦:创始人vs投资人,理想主义vs资本逻辑
- 李明 ←→ 林悦:务实的CEO和精明的投资人,目标一致但方式不同

**竞争关系**:
- 陈默团队 ←→ 张云团队:技术vs速度,理想vs现实,创新vs复制

**师徒关系**:
- 王教授 → 陈默:精神导师,学术支撑,理想的守护者

**利益关系**:
- 早期团队(周萌等):理想与现实的夹缝中,每个人都要做出选择
- 投资人(David等):不同背景的投资人,代表不同的价值取向

这个人物网络交织出科技创业的众生相:有坚持理想的,有务实妥协的,有背叛的,有坚守的。没有绝对的好人坏人,每个人都在自己的立场上做出选择,然后承担代价。这就是真实的创业世界。
