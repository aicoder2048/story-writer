# 人物小传

## 主要角色

**李渡（女主）**

28岁，眼神疏离的独立摄影师，总是穿黑色或深灰色的衣服，留着及肩的黑发，素颜，瘦削，有一双看透世事的眼睛。说话简短直接，很少笑，笑起来也带着点嘲讽意味，像是在看一场荒诞剧。习惯性抽烟，喜欢深夜拍照，独来独往。她不相信任何人的"一辈子"，因为父亲出轨毁了家庭，前男友劈腿她最好的朋友。她自我保护的方式就是不投入感情，永远保持抽离。遇到陈端成后，她一开始只是把他当临时居所，享受他的好但拒绝承认感觉。她从不说"我爱你"，只会说"你对我很好"。她准备出国逃避，但最终在国外的三个月里发现自己其实早就爱上了他。她的转变不是突然的，而是在无数个想起他的夜晚里慢慢醒悟——逃避不是保护自己，而是在折磨自己。最后她主动回国，说出"我想试试相信你"，这对她来说已经是最大的勇气。

**陈端成（男主）**

34岁，建筑设计公司老板，身材高大，总穿着熨帖的衬衫和西裤，戴着金丝边眼镜，外表斯文冷淡。性格成熟稳重，理性克制，生活规律自律，不擅长表达情感，但遇到李渡后整个人都变了。他第一次见到她就被吸引——不是因为她多美，而是因为她眼神里的那种洒脱和疏离，像一只永远抓不住的猫。他明知道她不相信爱情，明知道她会离开，还是忍不住对她好，给她做早餐，给她钥匙，说"我爱你"。他的深情不是轰轰烈烈的，而是细水长流的陪伴和包容。她说不信一辈子，他就说"你欠我一辈子"，用这种方式把她留在身边。她走后，他没有纠缠没有联系，只是默默等待，因为他相信她会回来——不是相信她会改变，而是相信她心里也有他。他的爱很成熟，知道爱一个人不是要改变她，而是接受她的所有，包括她的恐惧和逃避。

**方琪**

30岁，时尚杂志编辑，李渡的大学同学兼闺蜜，已婚，丈夫是律师。她是典型的都市女性，会打扮，爱八卦，话多热情，和李渡的性格完全相反。她是李渡唯一能说上几句心里话的人，也是唯一知道李渡前男友出轨细节的人。她一直担心李渡的情感状态，觉得她把自己封闭得太严实。当李渡和陈端成在一起后，方琪是第一个发现李渡其实动心了的人——她能看出李渡提到陈端成时眼神的变化，虽然李渡自己不承认。她劝过李渡不要逃避，说"你有没有想过，你其实也爱他"，但李渡当时听不进去。李渡在国外时，方琪会偶尔和陈端成联系，告诉他"她在伦敦一切都好"，也会旁敲侧击地问陈端成"你还在等她吗"。她是见证者，也是助攻者，相信他们最终会在一起。

## 重要配角

**李父**

55岁，某国企中层管理，已再婚。李渡12岁时出轨下属，和李母离婚后迅速再婚，还生了个儿子。他当初信誓旦旦说爱李母一辈子，婚姻破裂后对李渡的愧疚就是每个月多给点生活费。李渡成年后就不怎么联系他了，偶尔过年见一面也是客套。他是李渡不相信"一辈子"的第一个原因——父亲的背叛让她从小就明白，誓言是最不可靠的东西。

**李母**

53岁，小学老师，离异后独自抚养李渡长大。她是个温柔坚韧的女人，离婚后没有怨天尤人，而是把所有精力放在女儿身上。但她情感上的保守和封闭也深深影响了李渡——她告诉李渡"不要轻易相信男人"，教会李渡独立，却也无意间让李渡在情感上过度防备。她现在一个人住，偶尔和李渡视频通话，知道女儿和陈端成的事，但她不评价，只是说"你自己决定"。

**张轶（李渡前男友）**

29岁，广告公司创意总监。李渡23-26岁时的男友，当初追求李渡时很浪漫，说要娶她，但三年后劈腿了李渡当时最好的朋友。分手时他还试图解释"我们只是不合适"，被李渡一个耳光打回去。他是李渡不相信"一辈子"的第二个原因——连说要娶她的人都能背叛，她还能信谁？李渡出国前在机场偶然遇到过他，他已经结婚了，妻子不是那个当年的朋友。她看着他，突然觉得可笑——曾经以为天塌了的事，现在不过是一个陌生人。

## 其他角色

**老吴**：60岁，陈端成公司的老工程师，话不多但看人很准。他看出陈端成对李渡的感情，私下劝过陈端成"别太把自己陷进去"，但陈端成笑着说"已经陷进去了"。李渡走后，老吴会偶尔约陈端成喝酒，陪他度过最难熬的那段时间。

**Emma**：26岁，李渡在伦敦认识的英国女孩，同样学摄影。开朗健谈，和李渡成了朋友。她无意中听李渡提起陈端成，建议她"如果喜欢就回去啊，人生那么短，干嘛要逃避"。她是李渡回国的一个小小推动力。

**那对老夫妻**：结婚五十年的伦敦夫妇，在摄影展上偶遇李渡。他们相互扶持的样子让李渡第一次相信——也许真的有人能陪另一个人一辈子。老太太的那句"因为我们都相信，对方会陪自己一辈子"成了李渡的转折点。
