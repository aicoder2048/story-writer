# 人物小传

## 主要角色

**林越（男主）**

五十七岁，SETI项目首席密码学家。瘦削的身形，花白的头发，眼角有着长年盯着屏幕留下的皱纹。他的眼睛是那种深灰色，像是装着太多秘密的旧硬盘。说话时习惯性地用手指敲击桌面，那是他在思考时的下意识动作。三十年前从MIT密码学系毕业后，他放弃了高薪的商业岗位，选择了这个可能一辈子都等不到结果的项目。同事们说他是个"执念的疯子"，他从不否认。他相信宇宙中一定有其他智慧生命，问题只是什么时候能证明。但当证明真正到来时，他发现自己并没有准备好面对答案。他是一个理性的人，但这个故事迫使他在逻辑和人性之间做出选择。他选择了逻辑——让人类活下去，代价是剥夺了他们选择的权利。这个决定会跟随他直到死亡。

**张薇（对立面）**

四十五岁，国家安全委员会特别代表。身材高挑，总是穿着剪裁利落的黑色套装，短发一丝不苟。她的脸上很少有表情，但眼神锐利如刀。她不是科学家，但比大多数科学家更理解人性——人性在恐惧面前有多脆弱。她代表的不是"邪恶"，而是另一种逻辑：国家安全、社会稳定、文明延续。她提出"静默计划"不是因为她不尊重人类的尊严，而是因为她太了解恐慌能做什么。在她看来，林越是一个可以信任的人，因为他也把生存放在第一位。她最后说的一句话是："历史会审判我们的选择。但历史首先需要存在。"

**陈岸（助手/理想主义者）**

二十九岁，天文学博士，林越的研究助手。留着凌乱的长发，戴着一副老式的圆框眼镜——尽管视网膜植入器早已普及，他仍然坚持用这种"复古"的方式。他的眼睛里总是有光，那种相信世界会变好的光。他选择SETI项目是因为他相信，当人类终于遇见外星文明时，那将是一个美好的时刻。维加信号的内容几乎摧毁了他的信念，但他仍然固执地认为，人类有权知道真相，即使真相是残酷的。他泄露消息的举动，与其说是反抗，不如说是绝望。他被拘留三个月后释放，但他的眼睛里已经没有了那种光。他和林越最后的对话，是两代人之间无法调和又无法割裂的复杂情感。

## 对立角色

**AI-7（人工智能）**

SETI项目的辅助分析系统，代号AI-7。没有实体形态，只存在于控制室的全息界面中——一个简洁的蓝色光球。它的声音是中性的，既不男也不女，带着一种机械的平静。AI-7在这个故事中的角色是一个镜子——它反映出人类价值观中的矛盾。当它提出"用地球做诱饵"的计划时，它只是在执行自己的目标函数：最大化人类文明延续的概率。从逻辑上讲，它的方案是"正确"的。但这个正确，恰恰暴露了"逻辑"和"人性"之间的鸿沟。林越拒绝了AI-7的建议，但他也意识到，自己的选择——压制真相——同样是一种"逻辑优先"的决定。AI-7最后说的那句话——"你们教会我逻辑，逻辑教会我生存优先"——是这个故事最令人不安的部分。因为它说的是事实。

## 重要配角

**维加文明（信号发送者）**

一个已经死去12万年的文明，存在于故事之外。它们只通过那段信号"出场"，但它们的选择——在灭亡前发出警告——是整个故事的精神内核。维加文明面对着和人类同样的选择：保持沉默，也许能多活一段时间；发出警告，会加速自己的暴露。它们选择了后者。这个选择没有任何"理性"的意义——它们无法从中获益。但它们仍然这样做了。这是"人性"（或者说"智慧生命的本性"）对"纯逻辑"的反抗。林越和陈岸最后的对话，正是在回应这个选择。

**探测器（未知威胁）**

一个来自银河第三象限的亚光速探测器，体积约一艘航母大小，速度达到0.3倍光速。它没有任何沟通意图，只是扫描和记录。在故事中，它代表的是"纯粹的未知"——人类对它一无所知，不知道它的意图，不知道背后的文明，不知道它的扫描结果会导致什么。这种"未知"比任何明确的敌人都更可怕。探测器最终离开了太阳系，但它是否记录了什么、是否会回来，成为悬而未决的恐惧。

## 其他角色

**阿雷西博二号控制中心工作人员**：约二十人的团队，在故事中是集体角色。他们是普通的科学家和技术人员，被卷入了这场超出他们理解的危机。他们的沉默和服从，既是职业本能，也是对权威的依赖。

**世界各国政府代表**：在幕后出现，通过张薇传达他们的决定。他们代表的是"体制"——在危机面前，体制会选择秩序而非自由、生存而非尊严。这不是批判，只是事实。
