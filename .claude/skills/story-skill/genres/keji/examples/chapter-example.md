# 《代码与梦想》第一章:车库里的梦想

凌晨3点47分,陈默敲下了最后一行代码。

```python
if confidence > 0.95:
    return result
```

回车。编译。运行。

Terminal里滚动着白色的日志,最后定格在一行绿色的字:

`Test passed. Accuracy: 95.3%`

他盯着那个数字,手开始发抖。不是因为恐惧,是因为一个技术宅面对突破时,压抑不住的兴奋。

九十五点三。

比论文里承诺的95%还高。比顶尖的人类医生准确。比任何现存的AI诊断系统都强。

他转身,看向另一张桌子上趴着的李明。他的搭档、室友、这家还不存在的公司的联合创始人,正用手臂当枕头,睡在一堆乱七八糟的商业计划书上。

"李明。"陈默的声音有点哑,他已经48小时没睡了。

"嗯?"李明迷迷糊糊地抬起头,脸上还印着Excel表格的痕迹。

"成了。"

李明瞬间清醒,三步并作两步冲到陈默的电脑前。他盯着那行绿色的字,整整五秒钟没说话,然后突然抱住陈默:"我就知道!我他妈就知道你能做到!"

陈默被他勒得喘不过气,推开他:"别激动,这只是第一步。"

"第一步?这是最关键的一步!"李明在车库里转圈,手舞足蹈,"你知道这意味着什么吗?意味着我们可以去融资了!意味着我们可以注册公司了!意味着——"

"意味着我们可以救人了。"陈默打断他。

李明停下来,看着他,然后笑了:"对。意味着我们可以救人了。"

---

这是斯坦福大学校外三公里的一个老车库。月租800美元,水泥地面,墙上的油漆都在剥落,但采光不错,网络也快。过去六个月,这里就是他们的全世界。

左边是陈默的工作区:三台显示器,机械键盘,一堆空的红牛罐子。中间那块白板上写满了密密麻麻的公式和流程图,有些已经被擦掉重写过无数次。右边是李明的战场:一台MacBook,一沓打印出来的VC名单,还有三个版本的商业计划书。

墙上贴着一张破旧的海报,是乔布斯在斯坦福的演讲照片。下面有一行手写的字:

"Stay hungry. Stay foolish."

那是六个月前,他们租下这个车库的第一天,李明贴上去的。当时陈默说:"这太老套了。"李明说:"老套的才是经典。"

现在,看着这行字,陈默想:也许我们真的够foolish。辍学、创业、all in一个还不知道能不能变现的技术。他的父母,国内的中学教师,知道儿子放弃了斯坦福CS博士的offer时,母亲在电话里哭了。

"妈,我在做一件重要的事。"他当时说。

"什么事比读书更重要?"

"救人。"

电话那头沉默了很久,然后母亲说:"那你要小心。"

---

"我们需要demo。"李明说,他已经切换到了CEO模式,"一个能给投资人看的demo。"

"现在这个不行吗?"陈默指着屏幕。

"这是命令行界面,陈默。投资人看不懂这个。我们需要一个有界面的、能上传图片的、3秒出结果的APP。"

"那得两周。"

"我们有两周。"李明翻开笔记本,"我约到了林悦。"

陈默抬头:"谁?"

"知名VC合伙人,投资过三家独角兽,专注healthcare和AI。我给她发了BP,她回复了——'有意思,下周三见面'。"

"下周三?"陈默看了眼日历,"那是9天后。"

"所以你有9天时间做出demo。"李明拍拍他的肩膀,"我知道你能做到。"

陈默盯着他:"你是疯了还是疯了?"

"我是饿了。"李明笑,"你也是。我们账上还有多少钱?"

陈默打开银行账户,余额:$3,247.83。

"够吃两周泡面。"他说。

"够了。"李明站起来,伸了个懒腰,"吃完泡面,我们就是百万富翁。"

"或者是负债累累的失败者。"

"那也比在实验室里发论文强。"李明认真地说,"至少我们试过。陈默,你的技术是真的能救人的。论文发表要两年,专利申请要三年,然后呢?被锁在某个大公司的实验室里,永远看不到天日?我不想等那么久。"

陈默没说话。他想起王教授的话。

三周前,他去找导师,说自己想辍学创业。王教授摘下眼镜,擦了擦,说:"你确定?"

"确定。"

"为什么?"

"因为我看到了可能性。"陈默指着电脑上的数据,"这个算法,如果能应用到真实场景,每年能救数十万人。但如果走学术路线,从论文到临床应用,至少要五年。我等不了那么久。"

王教授沉默了很久,然后说:"去吧。学术界会一直在这里等你。但改变世界的机会,可能只有一次。"他顿了顿,"但陈默,记住一点:技术是中立的,但用技术的人不是。商业世界会给你很多诱惑,也会给你很多压力。坚持你的原则。"

"什么原则?"

"做正确的事,而不仅仅是赚钱的事。"

---

接下来的九天,陈默几乎没睡。

他用React搭了个简单的前端界面,用Flask做后端API,把训练好的模型封装成服务。界面很简陋:一个上传按钮,一个"开始诊断"的button,一个显示结果的区域。但它能用。

第七天凌晨,他完成了第一版。测试:上传一张肺部CT,点击诊断,loading 3秒,结果出现:

```
检测到2个可疑结节
位置:左肺上叶、右肺下叶
置信度:98.3%、94.7%
建议:进一步检查
```

他盯着屏幕,心跳加速。这是第一次,他的算法有了一张"脸"。不再是冰冷的命令行,而是一个真正的产品。一个医生可以用、病人可以信任的产品。

"李明。"他叫醒正趴在键盘上睡觉的搭档。

"又成了?"李明揉着眼睛。

"你看。"

陈默操作了一遍demo。上传、诊断、结果。3秒。

李明盯着屏幕,嘴角慢慢上扬:"完美。"

"不,还差得远。"陈默皱眉,"界面太丑了,用户体验很糟,而且——"

"陈默。"李明打断他,"这不是你的毕业设计,这是一个demo。它的目的是让投资人看到价值,而不是完美。你已经做得够好了。"

"可是——"

"没有可是。"李明合上电脑,"现在,我们去睡觉。明天,我们去改变世界。"

---

下周三,早上9点,Sand Hill Road,一栋玻璃幕墙的写字楼。

陈默和李明站在电梯里,陈默不停地整理着衬衫领子。他很少穿正装,这件衬衫还是李明借给他的。

"别紧张。"李明说。

"我没紧张。"

"你的手在抖。"

陈默把手插进口袋:"这不是紧张,是兴奋。"

"那就兴奋地展示我们的demo,然后兴奋地拿到500万美元。"

电梯门开了。

林悦的办公室很大,落地窗外是整个硅谷。她坐在办公桌后,手里拿着iPad,正在看什么。看到他们进来,她站起来,握手,简短地寒暄,然后说:

"开始吧。你们有15分钟。"

李明看了陈默一眼,陈默深吸一口气,打开电脑,连上投影。

"我们在做AI医疗诊断。"他说,声音比预想的平稳,"核心是一个深度学习算法,可以分析CT影像,识别早期癌症。"

他点开demo,上传了一张肺部CT——这是他从公开数据集里找的真实病例,已经做了脱敏处理。

点击"开始诊断"。

Loading条跳动。3秒。

结果出现:红色的圈标注在两个位置,置信度分别是98.3%和94.7%。

"这个病例,"陈默说,"三个月后确诊为早期肺癌,正是我们标注的这两个位置。"

林悦没说话,她站起来,走到屏幕前,盯着那两个红圈看了十几秒。

"准确率?"她问。

"95.3%。"

"样本量?"

"5000例CT影像,覆盖肺癌、肝癌、胃癌三个病种。"

"竞争对手?"

"Google在做,IBM也在做,但他们还没有产品上线。"

"你们的壁垒?"

"核心算法,我在斯坦福AI实验室开发的,已经申请了临时专利。"

林悦转身,看着他:"为什么辍学?"

陈默愣了一下。这个问题不在他准备的范围内。

"因为……"他组织语言,"因为如果走学术路线,从论文到临床应用,至少要五年。我等不了那么久。每年有数百万人死于癌症,很多是因为发现太晚。如果我的技术能提前发现,能救人,那为什么要等五年?"

"为什么不是三年、两年,而是'等不了'?"

陈默看着她,认真地说:"因为每一天的等待,都可能意味着有人因为漏诊而死。我不能接受这个。"

办公室里静了三秒。

然后林悦说:"商业模式?"

李明接过话:"To B,卖给医院和体检中心,按使用次数收费。初期免费试用,建立口碑,六个月后开始收费。"

"市场规模?"

"仅中国,每年CT检查超过5000万次,如果渗透率达到10%,每次收费50元,年收入2.5亿。"

"团队?"

"我和陈默,一个技术一个商业。计划A轮融资后,扩张到15人,包括算法工程师、产品经理、销售。"

林悦重新坐下,在iPad上记着什么。然后她说:

"你们需要多少钱?"

李明和陈默对视了一眼,李明说:"500万美元,出让20%股份。"

"用途?"

"技术研发300万,市场推广100万,团队建设100万。"

林悦点点头,然后说:"我给你们两个选择。第一,500万美元,20%股份,但有对赌协议:一年内必须达到10万用户,否则我有权按原价回购股份。第二,300万美元,15%股份,没有对赌,但你们要接受我推荐的CFO。"

陈默的心跳加速。这是真的。投资人真的要投他们。

李明说:"能给我们一周考虑吗?"

"三天。"林悦说,"三天后,我要答复。"她站起来,握手,"做得很好。尤其是你,陈默。我见过很多技术创业者,但很少有人像你这样,真正关心技术能救什么人,而不只是能赚多少钱。"

---

走出办公楼,陈默和李明在停车场里站了很久,一句话都没说。

然后李明突然吼了一声,在空旷的停车场里回响。

"我们他妈要拿到投资了!"

陈默笑了,笑得眼泪都出来了。不是因为钱,是因为终于有人相信他们。相信这个在车库里折腾了六个月的项目,相信这个还不知道能不能改变世界的梦想。

"走,吃饭去。"李明说,"今天我们奢侈一把,不吃泡面了。"

"吃什么?"

"In-N-Out。"

"还是泡面级别的。"

"那是庆祝版的泡面。"

两个人笑着走向李明的二手本田,车后座还堆着没洗的衣服和空的咖啡杯。发动机发出嘶哑的声音,但终于启动了。

开出停车场,陈默看着窗外飞驰而过的风景。硅谷的天空很蓝,阳光洒在玻璃幕墙的大楼上,闪闪发光。这里诞生过苹果、谷歌、Facebook,诞生过无数改变世界的公司。

也许,有一天,他们也会是其中之一。

"李明。"他说。

"嗯?"

"我们真的要做到。"

"做到什么?"

"改变世界。"

李明握着方向盘,笑了:"那是必须的。否则我们辍学干嘛?"

车子驶向夕阳,驶向未知的未来。在车库里,白板上的公式还没擦,电脑还亮着,屏幕保护程序显示着一行字:

"Stay hungry. Stay foolish."

这只是开始。

真正的故事,才刚刚开始。

**写作分析**:

**节奏控制**:
- 开篇直接进入技术突破场景,3分钟建立创业氛围
- 快节奏对话推进情节,展现人物性格
- 多个时间切换(现在-回忆-时间推进),保持紧凑感

**技术细节**:
- 真实的代码片段和技术术语(Python, React, Flask)
- 可信的准确率数据和测试流程
- 专业的融资谈判和商业术语

**人物塑造**:
- 陈默:技术天才,理想主义,通过细节(敲代码、揉太阳穴)展现
- 李明:商业头脑,现实主义,通过对话和行为展现
- 林悦:专业投资人,通过连续提问和冷静判断展现

**情感克制**:
- 不直接写"很激动",而是写行为(李明转圈、陈默笑到流泪)
- 重要时刻用沉默和留白传递情感
- 用细节(二手本田、In-N-Out)展现创业的真实

**主题呼应**:
- "做正确的事,而不仅仅是赚钱的事"贯穿全章
- "Stay hungry. Stay foolish"作为精神符号
- 为后续冲突(理想vs现实)埋下伏笔
