# 人物小传

## 主要角色

**林初夏（女主）**

高二文科班学生，十七岁。留着齐肩的黑色长发，总是扎成简单的马尾，刘海有点长，常常遮住眼睛。戴一副细框眼镜，镜片后面是一双敏感细腻的眼睛。她长得不算特别出众，但笑起来有两个浅浅的酒窝。平时穿着很朴素，大多是白色衬衫配牛仔裤，很少化妆，但总是把自己收拾得干干净净。

性格内向敏感，在班里存在感不强，但成绩一直是年级前十。她不太擅长交际，除了苏小沫，没什么其他朋友。她习惯把心事藏在心里，不轻易表露情感。她很善良，总是为别人着想，却常常忽略自己的感受。她喜欢写日记，喜欢看夕阳，喜欢一个人发呆。

她从高一开始就暗恋陆之昂，三年来默默关注他的一切。她知道他的作息时间，知道他喜欢什么，却从来没有勇气说出来。她的暗恋是小心翼翼的，生怕被人发现，生怕打扰到他。她觉得自己不够好，配不上那么优秀的他，所以只能远远看着。

这个故事是她的成长之旅。从一开始的胆怯退缩，到最后勇敢说出心意，她学会了正视自己的情感，学会了不让自己后悔。虽然最终没能和陆之昂在一起，但她不再是那个只会躲在角落里的女孩了。她变得更勇敢，也更懂得爱自己。

**陆之昂（男主）**

高二理科班学生，十八岁。身高一米八，身材修长，长得很俊朗——剑眉星目，鼻梁高挺，皮肤白净。他总是穿得很整洁，校服也熨得笔挺，头发梳得一丝不苟。他看起来有点像韩剧男主角，在学校里有不少女生暗恋他。

性格温和有礼，待人接物很有分寸感，对谁都很nice。成绩是年级第一，但从不显摆，反而很谦虚。他在学校里人缘很好，老师喜欢他，同学也尊敬他。他看起来很完美，像是那种"别人家的孩子"。

但其实陆之昂并不像表面看起来那么快乐。他的父母对他期望很高，他一直在努力成为他们想要的样子。他很少有真正放松的时候，总是在扮演"好学生""好儿子"的角色。他表面温柔，内心其实很孤独，缺乏真正理解他的人。

他喜欢苏小沫,是因为她的开朗和真实吸引了他。和苏小沫在一起的时候，他可以放下那些伪装，做回真正的自己。他对林初夏一直心怀愧疚——他其实早就发现了她的暗恋，但他选择装作不知道，因为他不想伤害她，却又不知道怎么回应。当林初夏把信给他的时候，他很震撼,也很感动。他第一次认真审视自己的情感，也第一次明白，有时候拒绝也是一种尊重。

**苏小沫（闺蜜/情敌）**

高二文科班学生，十七岁。长得很漂亮——大眼睛，双眼皮，瓜子脸，笑起来眼睛弯弯的，很甜美。她喜欢穿颜色鲜艳的衣服，总是把自己打扮得很精致。留着一头自然的长卷发，发梢带着淡淡的栗色。

性格开朗活泼，是班里的文艺委员，歌唱得很好,也很会跳舞。她人缘很好，在学校里是个小明星式的人物。她很热情，对朋友很真诚，是那种会为朋友两肋插刀的性格。

苏小沫和林初夏从初中就是最好的朋友。她很喜欢林初夏，觉得她善良单纯，一直把她当最好的闺蜜。她完全不知道林初夏也喜欢陆之昂，在和陆之昂在一起后，还兴奋地和林初夏分享甜蜜的细节，丝毫没有察觉林初夏的痛苦。

当她最后知道真相时，她非常愧疚和自责。她哭着和林初夏道歉，说如果早知道她也喜欢陆之昂，她一定不会接受他的告白。但林初夏安慰她，说这不是她的错。她们的友情在这次考验后，反而变得更加珍贵。苏小沫从这件事中学会了更加细心地关注朋友的感受，也更加珍惜林初夏这个朋友。

## 重要配角

**叶修齐（好友）**

高二理科班学生，陆之昂的死党，十八岁。长得不算帅，但很阳光，总是笑嘻嘻的。身高一米七五，有点胖,喜欢运动，是校篮球队的成员。他性格大大咧咧，很会开玩笑，是班里的开心果。

他暗恋苏小沫很久了，但一直没敢说。当陆之昂和苏小沫在一起后，他虽然失落，但还是选择默默祝福。他发现了林初夏的秘密，因为他自己也在经历同样的痛苦。他理解那种喜欢一个人却不敢说的感觉，所以他鼓励林初夏勇敢一次。

他是个很善良的人，虽然表面上嘻嘻哈哈，但心思很细腻。在林初夏最痛苦的时候，是他陪在她身边，给她支持和鼓励。他对林初夏说的那句话——"有些人就是这样,总觉得自己的喜欢不重要"——其实也是在对自己说。

在这个故事里，他代表了另一种青春——那种默默付出、不求回报的喜欢。他和林初夏是同道中人，都在成长中学会了放手和释怀。

**赵老师（班主任）**

三十岁出头的年轻班主任，教语文。戴一副黑框眼镜，看起来很斯文，说话温和有耐心。她是那种很理解学生的老师，不会一味地强调学习，也关心学生的情感和心理健康。

她发现了林初夏的异样——成绩下滑、上课走神、眼睛总是红红的。她找林初夏谈话，没有直接问发生了什么,只是告诉她："青春期有心事很正常，老师也经历过。重要的是，不要让这些事情成为你的负担。该说的话就说出来，该放下的就放下。人生很长，高中只是一个阶段。"

她的话给了林初夏很大的勇气。她让林初夏明白，感情不是羞耻的事情，勇敢表达也不是错误。她在关键时刻给了林初夏支持，让她有勇气做出最后的选择。

## 其他角色

**林初夏的父母**：普通的工薪家庭，父亲是公司职员，母亲是小学老师。他们对林初夏的要求不高，只希望她健康快乐。但林初夏很懂事，不想让他们担心，所以总是把心事藏在心里。

**陆之昂的父母**：事业有成，对儿子期望很高。他们给陆之昂最好的教育资源,但也给了他很大的压力。陆之昂一直在努力成为他们期望的样子，却很少问自己真正想要什么。

**文科班同学们**：是故事的背景群像，代表了普通的高中生活。他们有的在为高考焦虑，有的在谈恋爱，有的在追逐梦想。他们共同构成了青春校园的真实氛围。

**理科班同学们**：陆之昂的同班同学,崇拜他的成绩，也羡慕他的爱情。他们是故事中的旁观者，见证了这段青春的美好和遗憾。
