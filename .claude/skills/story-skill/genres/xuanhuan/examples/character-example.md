# 人物小传

## 主要角色

**萧炎（男主）**

一袭黑色劲装，身形挺拔，眉宇间带着一股不屈的傲气。他的眼神深邃如潭，经历过从天才到废物再到强者的蜕变，早已磨练出钢铁般的意志。左手无名指上戴着一枚古朴的黑色戒指，那是父亲留下的唯一遗物，也是他最大的秘密——里面藏着药老的灵魂。

萧炎曾经是萧家最耀眼的天才，九岁凝聚斗气，十一岁达到斗之气九段，被誉为百年难遇的修炼奇才。然而十二岁那年，他的斗气突然神秘消失，从云端跌入谷底。三年间，他忍受着族人的冷眼、嘲笑和羞辱，但从未放弃，每天都在尝试修炼，即使毫无成效。

他的性格因这三年的经历变得沉稳坚韧，不轻易表露情绪，但内心却燃烧着不屈的火焰。他可以忍受屈辱，但绝不会忘记仇恨。当纳兰嫣然当众退婚时，他没有暴怒，只是平静地说出"三十年河东三十年河西，莫欺少年穷"，这句话成为他逆袭的宣言。

遇到药老后，萧炎展现出惊人的修炼天赋和坚韧意志。炼化青莲地心火时，常人早已承受不住剧痛而放弃，他却死死咬牙坚持了一天一夜。他对敌狠辣果决，不留后患；对友重情重义，愿为兄弟两肋插刀；对恩人感恩图报，答应帮药老复仇绝不反悔。

他的战斗风格霸道凶猛，擅长以异火克敌制胜。他的装逼风格低调内敛，不主动炫耀，但被人挑衅时会以绝对实力碾压，让对手震惊。他最经典的台词就是"三十年河东三十年河西，莫欺少年穷"，这句话伴随他一路成长，从废物逆袭为强者。

**萧薰儿（女主）**

一袭紫裙，容颜绝美，气质高贵神秘。她看似娇弱，实则是来自远古八族之一古族的天之骄女。她从小就被派到萧家守护萧炎，因为萧炎体内流淌着古族的血脉，是古族未来的希望。

她一直默默守护着萧炎，在他最落魄的三年里，只有她从未改变对他的态度，依然称他"萧炎哥哥"，依然相信他会重新站起来。她知道萧炎斗气消失的秘密，也知道戒指中的药老，但她从不说破，只是默默陪伴。

她的实力远超常人想象，看似只是斗者，实际上已是斗王。她的背后有古族强者守护，一旦萧炎遇到生命危险，那些强者就会出手。她对萧炎的感情真挚深沉，但从不表露，只是用行动默默支持他。

**药老（导师）**

原名药尘，曾经是炼药师公会的会长，九品炼药师，斗尊巅峰强者。他掌握着异火榜排名第二的骨灵冷火，炼药术通天，在大陆上享有盛名。然而十五年前，他被魂殿围攻，肉身被毁，只剩灵魂藏身于纳戒中。

这枚纳戒机缘巧合下被萧炎父亲萧战得到，作为传家宝给了萧炎。药老为了维持灵魂不灭，不得不吸收萧炎的斗气，这导致萧炎的斗气"消失"。但药老心中有愧，也看中萧炎的天赋和意志，决定倾囊相授，传他炼药术和强大功法。

药老性格洒脱豁达，虽然身为斗尊强者，但对萧炎这个小辈却没有架子，亦师亦友。他经验丰富，智慧过人，常常在关键时刻给萧炎指点迷津。他对萧炎要求严格，但也处处为他着想，可以说是萧炎成长路上最重要的导师。

他最大的愿望就是恢复肉身，然后向魂殿复仇。为此，他需要收集炼制复活丹的材料，其中就包括异火。这也是他让萧炎收集异火的原因之一。

## 反派角色

**纳兰嫣然（主要反派）**

云岚宗年轻一代第一天才，容貌出众，气质高贵。她自幼修炼，天赋卓绝，深得师父云山喜爱。三年前她已是斗者九星，如今更是突破到大斗师后期，掌握地阶斗技。

她出身纳兰家，与萧家有婚约。但当萧炎沦为废物后，她觉得这样的废物不配做她的夫婿，会拖累她的修炼之路。于是她亲自登门，当众退婚，丝毫不顾及萧炎的颜面。

她性格高傲，自视甚高，认为废物就是废物，永远不可能翻身。她根本没把萧炎的"三年之约"当回事，只是觉得给他一个机会证明自己，三年后再踩他一次而已。

三年后的决战中，她被萧炎的实力震惊，尤其是看到异火时，她才明白自己当年是多么傲慢和短视。最终她被萧炎击败，输得一败涂地，只能眼睁睁看着萧炎说出"当年你对我爱答不理，今日我让你高攀不起"这样的打脸台词。

**云山（最强反派）**

云岚宗宗主，斗宗强者，在加玛帝国几乎无敌的存在。他须发皆白，仙风道骨，实力深不可测。纳兰嫣然是他最得意的弟子，他视若己出。

当纳兰嫣然被萧炎击败时，他恼羞成怒，不顾身份对小辈出手，想要击杀萧炎。这暴露了他虚伪的本质——平时道貌岸然，关键时刻却不择手段。

他被药老拦下，两大强者对峙。他认出药老后惊恐万分，因为当年他也参与了围攻药老的行动，是魂殿的帮凶之一。这个秘密的揭露，让他在加玛帝国的声望大跌。

## 重要配角

**萧战**

萧炎的父亲，萧家族长，大斗师实力。他对儿子的期望很高，当萧炎沦为废物时，他失望但从未放弃。他相信儿子总有一天会站起来。当萧炎重新崛起时，他比谁都高兴，眼中甚至含着泪水。他是萧炎最坚实的后盾，无论何时都支持儿子。

**加刑天**

加玛帝国皇帝，斗皇巅峰强者。他慧眼识英才，看中萧炎的潜力，多次想招揽。虽然萧炎拒绝加入皇室，他也不勉强，反而赐予侯爵之位，与萧炎结交。他是一位明君，知道如何笼络强者。

**海波东**

冰皇海波东，斗皇强者，被魂殿诅咒困于加玛帝国。萧炎机缘巧合下帮他解除诅咒，两人结为好友。海波东对萧炎很是欣赏，在关键时刻会出手相助。

**美杜莎女王**

蛇人族女王，斗皇巅峰强者。她美艳绝伦，但性格冷酷无情。她与萧炎有过一段纠葛，既是敌人又是朋友，关系微妙。

## 其他角色

**萧家弟子**：萧玉、萧宁等，曾经嘲笑萧炎，后来被萧炎打脸，羞愧不已。

**云岚宗弟子**：各种配角，用来衬托萧炎的强大。

**加玛帝国强者**：各大势力的代表，见证萧炎的崛起。
