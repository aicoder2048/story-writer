# 人物小传

## 主要角色

**林寒（男主）**

一袭青衫，身形修长，剑眉星目间透着一股清冷之意。左臂有一道旧疤，那是十年前血案留下的印记。他平日少言寡语，目光深邃如潭，总让人觉得他心事重重。然而一旦出剑，便如寒风骤起，凌厉无匹。他的剑法师承华山正宗，又得林家真传，刚柔并济，变化莫测。性格坚韧内敛，对敌冷酷无情，待友却是肝胆相照。十年的寄人篱下，让他学会了隐忍，也让他在心底压抑了太多的仇恨。他表面对师父岳不凡恭敬有加，心中却始终未忘灭门血仇。当他终于发现仇人就是养育自己的师父时，内心经历了巨大的挣扎。最终，他在绝境中悟道，明白了侠义的真谛不在于杀人复仇，而在于惩恶扬善、济世救人。他选择让仇人活着受辱，比一剑杀之更显侠者胸怀。

**苏婉儿（女主）**

峨眉派当代弟子中的翘楚，一身淡蓝衣裙，眉目如画，举止端庄。她的剑法灵动飘逸，以巧制胜，颇有峨眉风范。性格温婉善良，心思细腻，却绝非柔弱之辈。她有着侠女的果敢与决断，关键时刻绝不畏缩。她初遇林寒时，便看出他眼中的阴郁与孤独。一路同行，她渐渐被这个沉默寡言却重情重义的少年打动。在林寒最绝望的时刻，是她冒死相救，用行动诠释了"侠义"二字。她与林寒之间的情愫，含蓄而真挚，从未说破，却尽在不言中。"真正的侠者，岂能坐视不平之事？"这是她的信条，也是她与林寒能够并肩作战的根本原因。

**叶孤城（挚友）**

江湖游侠，无门无派，一柄长剑走天涯。他生得俊朗不羁，总是一身白衣，嘴角挂着玩世不恭的笑意。看似吊儿郎当，实则剑法精绝，内力深厚，是真正的高手。他性格豪爽洒脱，最重情义。与林寒一见如故，虽然相识不久，却愿意为他断后、冒死打探消息。他的口头禅是"人生难得一知己，千古知音最难寻"。他虽然表面玩世不恭，内心却有着自己的侠义准则。最后能够请来武林盟主为林寒作证，既是他广结善缘的结果，也是他为兄弟两肋插刀的义气体现。

## 反派角色

**岳不凡（主要反派）**

华山派掌门，年近五旬，须发灰白，面容清癯，一派仙风道骨的模样。他在武林中素有"君子剑"之称，行事光明磊落，受人尊敬。然而这一切都是伪装。十年前，他为了夺取林家的《寒梅剑谱》，灭了林家满门，只留下年幼的林寒，收为弟子，以掩人耳目。他城府极深，喜怒不形于色，就连收养林寒十年，也未露出半点破绽。他的武功以寒梅剑法为根基，招式阴狠毒辣，与他平日里的君子形象判若两人。当真相被揭穿时，他宁愿承受武林公敌的下场，也不肯认错。最终身败名裂，比死更难受。他的结局，正应了那句话："多行不义必自毙。"

## 重要配角

**无名老人**

隐居深山的绝世高手，须发皆白，仙风道骨。他是当年林家的故交，亲眼目睹了血案全过程，却因实力不济无法阻止。此后他潜心苦练，只为有朝一日能够帮助林家报仇。他的武功已臻化境，尤其对寒梅剑法有独到见解。他不仅传授林寒剑法，更教会他侠义的真谛："剑法的最高境界，不是招式的精妙，而是心境的通达。"他是林寒成长路上最重要的导师，却从不居功，传完武艺后便飘然而去。

**武林盟主**

一位德高望重的老前辈，为人公正，在江湖上有着极高的威望。他受叶孤城之邀，带领各派掌门前来华山作证。他的出现，代表着武林正义的力量。在他的见证下，岳不凡的罪行被公之于众，林家的冤案终于昭雪。他对林寒说："年轻人，你有一颗侠义之心，前途无量。"

## 其他角色

**华山师弟们**：林寒的同门师弟，武功各有千秋。他们大多不知真相，被岳不凡利用来对付林寒。真相大白后，他们羞愧不已，纷纷向林寒赔罪。

**老乞丐**：当年血案的目击者，知道真相却不敢说。在小镇上被三人寻访，刚要吐露真相便被飞刀杀害。他的死，让林寒更加坚定了查明真相的决心。

**黑衣蒙面人**：岳不凡的暗杀队，身穿黑衣，脸戴面具，专门负责清除知道真相的人。他们武功不弱，行事阴狠，是岳不凡见不得光的打手。
