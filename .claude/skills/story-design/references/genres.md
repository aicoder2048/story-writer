## Genre Characteristics Summary

| 类型 | 核心特点 | 典型调性 |
|------|----------|----------|
| 童话故事 | 纯真美好、寓言哲理 | 温暖治愈、奇幻冒险 |
| 武侠江湖 | 刀光剑影、侠义恩仇 | 快意江湖、侠骨柔情 |
| 科幻小说 | 科技想象、未来探索 | 硬核科幻、人文反思 |
| 都市言情 | 现代都市、情感纠葛 | 甜蜜治愈、虐恋成长 |
| 玄幻修仙 | 修炼体系、逆天改命 | 热血升级、仙侠奇缘 |
| 悬疑推理 | 谜团解密、逻辑推理 | 烧脑悬疑、暗黑惊悚 |
| 历史军事 | 历史背景、战争风云 | 金戈铁马、权谋争霸 |
| 古风权谋 | 宫廷斗争、权力博弈 | 深宫暗涌、智计交锋 |
| 青春校园 | 校园生活、青春成长 | 纯爱初恋、热血励志 |
| 现代言情 | 现代社会、爱情故事 | 甜宠日常、虐心虐恋 |
| 恐怖灵异 | 灵异事件、恐怖氛围 | 惊悚悬疑、细思极恐 |
| 末日生存 | 末日背景、生存挑战 | 废土求生、人性考验 |
| 科技创业 | 科技行业、创业故事 | 热血奋斗、商战风云 |
| 科普纪实 | 科学知识、纪实叙事 | 严谨求知、深度探索 |
| 医疗生命 | 医疗行业、生命伦理 | 医者仁心、生死抉择 |
