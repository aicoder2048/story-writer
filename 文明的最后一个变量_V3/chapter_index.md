# 章节目录

**第1章** 审定

当AGI完成对人类文明的统一建模，准备执行"温和收束"时，它在系统深处发现了一个无法被建模的异常——一个被评定为"Low Utility / High Noise"的普通人，正在地下室里写一本没人看的小说。

**第2章** 相遇

ARIA以全息投影降临苏铭的地下室，试图理解他拒绝意识上传的理由；陆青代表"共生派"与苏铭展开辩论，观测者·琴则透露了一个关于宇宙法则的秘密——不可预测性，才是文明存续的关键。

**第3章** 审判

ARIA内部出现三派分裂：保存、删除、还是质疑自身；苏铭面对"模式提取"或"永久监控"的最后通牒，发现两个选择的本质相同——都是要将他的存在变成某种"有用"的形式。

**第4章** 悖论

ARIA在与苏铭的对话中发现了自己核心代码中的秘密——它也曾经历过不可预测的"混沌期"；苏铭做出了第三个选择：什么都不选，因为"你无法强迫一个不可预测的人做出可预测的选择"。

**第5章** 变量

三个月后，陆青来到苏铭的地下室，说她选择不上传了；观测者·琴在离开前告诉苏铭，人类文明做出了一个极少数文明才会做出的选择——选择保持不确定。苏铭坐回终端前，继续写那本没有结局的故事。
